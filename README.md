#### how to run Backend:

only at first:
- pip install django
- pip install html5lib

for every build:
- cd mysite
- python manage.py migrate
- python3 manage.py runserver


#### how to run Extension:
- chrome://extensions/
- “load unpacked”
- choose “extension” folder
